# BloodConnect - Donor Registration System

## Project Description
A web application built with Flask to register blood donors and maintain a searchable database. This project demonstrates frontend validation, backend routing, and database integration.

## Technologies Used
- **Backend:** Python Flask
- **Database:** SQLite
- **Frontend:** HTML5, Bootstrap 5
- **Behavior:** jQuery (Validation & DOM Manipulation)
- **Version Control:** Git & GitHub

## Features
- Donor Registration Form with Validation
- Dynamic Form Fields (First-time donor logic)
- Donor List View with Bootstrap Cards
- Responsive Design
